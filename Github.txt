Github

仓库（Repository）
仓库用来存放项目代码，每个项目对应一个仓库，多个开源项目则有多个仓库；

收藏（Star）
收藏项目，方便下次查看；

复制克隆项目（Fork）
该fork的项目是独立存在的；

发起请求（Pull Request）

关注（watch）
关注项目，当项目更新可以接受通知；
